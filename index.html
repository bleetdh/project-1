LumberJack
instructions

Hello there! This is the Jack the LumberJack! I have found a really tall tree to cut. Life would have been easy if trees do not grow branches! So help me out by telling me to cut from the LEFT or RIGHT so as to avoid those irritating branches. Hurry hurry before the sun sets!

  • Use the LEFT and RIGHT arrow key to start the game and move Jack left or right.
  • Each correct move will add 50 to the timer.
  • Game will be over once the timer runs out or if Jack hits a branch.
  • Use the SPACE key to restart the game.
